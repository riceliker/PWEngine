# Ready Your Environment

## Ready Environment 
The lecture will use such software to build program. Use other is okay.  
IDE: vscode.     
Build Tool: xmake.   
Compiler: clang.   
Shell: zsh.   

## Create Workspace
Open the vscode. Start a new folder. Create such file:
1. create xmake.lua
```lua
add_rules("mode.debug", "mode.release")
add_rules("plugin.compile_commands.autoupdate", {outputdir = "."})
set_languages("c++23")
set_warnings("all")

add_requires("libsdl3", "libsdl3_image", "libsdl3_ttf", "glm", {system = false})

target("Demo") -- your project name
    set_kind("binary")
    add_files("src/**.cpp")
    add_includedirs("src", "include")

    add_packages("libsdl3", "libsdl3_image", "libsdl3_ttf", "glm")

    add_linkdirs("lib")
    add_links("PWEngine")

    set_targetdir("build/")

    set_symbols("debug")
    set_optimize("none")
    set_strip("none")
```
2. Create include folder. Copy folder from source `include`. 
it make be like that:  
-> PWECore.  
-> PWEType.   
-> PWEngine.hpp.   

3. Create a src file. You code will write here.

4. Open your terminal. `cd` your project folder. use `xmake` to build you project.
you will see a file in build. Clicked it to run.
