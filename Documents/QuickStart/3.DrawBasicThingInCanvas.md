# Draw Basic Thing In Canvas

## Set Background Color
We have not anything to provide to window. So we see nothing here.
Now. Let's send the window color to window.
```C++
string loop() override
{
    clearCanvas(PWEColor{255, 255, 255, 255});

    submitCanvas();
    return "";
}
```
`clearCanvas()` is a member function in class `PWECanvasScene`. It will clear canvas as a color.
```C++
void clearCanvas(PWEColor color)
```
`PWEColor` is a struct to description color. for example, use white color #FFFFFFFF
```C++
// red, green, blue, alpha
PWEColor(255, 255, 255, 255)
```
`submitCanvas()` is a member function in class `PWECanvasScene`. It will clear canvas as a color.
When use this function, the canvas image will submit for window.
