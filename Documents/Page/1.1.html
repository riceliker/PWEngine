1.1 Hello PWEngine

1.1.1 PWEngine, A free software engine

Welcome to use PWEngine. This web document is help you know something about PWEngine.

PWEngine is a GUI Engine and Game Engine. Basic by language is C and SDL framework.

PWEngine is a free library, use LGPL3.0 License. That's mean anyone can help engine developing, Can be free to copy, remove, add, compile, run the code.

PWEngine support multi-platform(Desktop is the main focus). But except the DRM platform.(such as iOS, Xbox, PlayStation, Switch).

Although the PWEngine is written by C. But we comment you choose C++ or Rust. It has memory manager and container.

The GUI Editor is optional. And we abandon the huge editor to do anything. Remember `Lite is best`.
