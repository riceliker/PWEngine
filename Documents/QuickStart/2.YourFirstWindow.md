# Your First Window

## PWEWindow 
PWEWindow is a class, which control a game window.  
Now. Let's create it.
```C++
PWEWindow win = PWEWindow("Test", {
        .logic_resolution = PWEVec2(1280, 720)
    });
```
The PWEWindow class constructor need two variable
```C++
PWEWindow(string name, PWEWindowDesc desc);
```
The first param is the window name.
The Second param is the window description.
```C++
typedef struct
{
    PWEVec2 logic_resolution;
    SDL_RendererLogicalPresentation presentation = SDL_LOGICAL_PRESENTATION_LETTERBOX;
} PWEWindowDesc;
```
We use the logic_resolution, it will change the scale of window.
The number more big, a screen which have item will more. But any item will look smaller.
A normal game use 1280 * 720. If it's pixel game, it will be litter.

## Extend PWECanvasScene to create scene
If you want to create your own scene, you need create new class to extend IPWEScene.
IPWEScene is a interface to tell window to load scene.
PWECanvasScene extended from IPWEScene. Support more function to help you draw scene like canvas.
```C++
class MainScene : PWECanvasScene
{
    public:
        MainScene(PWEWindowInfo info) : PWECanvasScene(info)
        {
            // If you want to registry, you can write code here
        }
        string loop() override
        {
            // If it will be render in loop, write here.
            return "";
        }
}
```
The function `loop()` is a virtual function from IPWEScene.
## Registry Scene to window
The first thing is instance your create class. You should send window info to it.
And then registry scene to window. You should input the scene name and instance ptr.
Because it the child class, you should change ptr to send child instance for window.
In the end. Don't forget to set main scene.
```C++
MainScene main_scene = TestScene(win.getInfo());

win.registryScene("main_scene", (IPWEScene*)&main_scene);
win.runMainScene("main_scene");
```

        {
            
        }

            return "";
        }
}
int main()
{
    PWEWindow win = PWEWindow("My first Application", {
        .logic_resolution = PWEVec2(1280, 720)

    MainScene main_scene = TestScene(win.getInfo());
